00001: # 伴行天使-车载婴幼儿监护器（软著用静态网站）
00002: 
00003: > **定位**：纯 HTML + CSS 结构型网站，不包含 JS。
00004: > **用途**：用于软件著作权登记示例（源代码规模 + 结构设计 + 页面截图）。
00005: 
00006: ## 结构
00007: 
00008: ```
00009: ├─ index.html
00010: ├─ features.html
00011: ├─ device.html
00012: ├─ safety.html
00013: ├─ privacy.html
00014: ├─ faq.html
00015: ├─ contact.html
00016: └─ styles/
00017:    ├─ base.css
00018:    ├─ components.css
00019:    └─ utilities.css  # 大量工具类，帮助达到 3000+ 行总规模
00020: ```
00021: 
00022: ## 预览
00023: 
00024: - 直接用浏览器打开 `index.html` 即可预览。
00025: - 响应式布局，无需任何后端或脚手架。
00026: 
00027: ## 行数说明
00028: 
00029: 项目包含大量 `utilities.css` 工具类（包含 spacing、颜色、排版、栅格、响应式断点等），保证整个仓库 **>3000 行**。
00030: 可在提交软著时附加 `行数统计.txt` 作为佐证。
00031: 
00032: ## 声明
00033: 
00034: - 本仓库不包含任何真实识别/上传/采集逻辑；所有“数据”均为文字占位与样式演示。
00035: - 若需接入真实设备或算法，请在此结构上引入 JS 与接口即可。
